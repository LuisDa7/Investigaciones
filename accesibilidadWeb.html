¿Qué es?

La accesibilidad web es el grado de simplicidad con el cual el usuario interpreta y comprende el contenido de una página web. Se puede concluir que la accesibilidad debe permitir que el usuario navegue entienda e interactue con todos los objetos del sitio web.

Importancia

La accesibilidad web permite que las personas, sin importar sus capacidades, puedan acceder y utilizar de forma sencilla el contenido del sitio web.

Eliminar las barreras para personas con discapacidades o con problemas de salud a la hora de navegar en la web es la verdadera importancia de la accesibilidad web.

Principios

Perceptible

Los usuarios deben ser capaces de acceder al contenido de la página de forma que ellos no necesiten ayuda para lograrlo.

Ejemplo

La página lenguajehtml.com Permite que los usuarios accedan a su contenido de forma sencilla. Una vez se ingresa, automáticamente se sobreentiende qué hacer y cómo buscar la información.

Operable

Cada objeto presentado en la página web debe ser operable y sencillo de utilizar. Es bueno agregar ayudas para que los usuarios se guíen mejor.

Ejemplo

La página uiverse.io Hace que las personas que ingresan a la página puedan interactuar directamente con el contenido, obteniendo así lo que buscan.

Comprensible

El contenido debe ser legible y comprensible, con una apariencia que permita al usuario entender y manipular la página.

Ejemplo

La página pixabay.com Permite que los usuarios comprendan el funcionamiento de la página, y como utilizarla, de una forma realmente fácil.

Robustez

La página debe contener la información suficiente para que pueda ser interpretada correctamente. La inclusión de sistemas de tecnología de asistencia hace más robusta la página web.

Ejemplo

La página www.w3schools.com Con su inmensa cantidad de documentación, esta página se vuelve muy accesible y confiable para los usuarios. Es una página en la que cualquiera podría buscar ayuda para programar.
